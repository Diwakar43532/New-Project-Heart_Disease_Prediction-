❤️ Heart Disease Prediction System
📌 Project Overview

This project is a Machine Learning based web application that predicts the risk of heart disease using patient health parameters. The application is built with Python, Scikit-learn, Streamlit, and GitHub and is deployed on Streamlit Community Cloud.

🚀 Features
Predicts heart disease risk in real time
User-friendly web interface
Uses Machine Learning model for prediction
Fast and accurate results
Cloud deployment using Streamlit
🛠️ Technologies Used
Python
Pandas
NumPy
Scikit-learn
Joblib
Streamlit
GitHub
📊 Input Parameters

The model uses the following features:

Age
Sex
Chest Pain Type
Resting Blood Pressure
Cholesterol
Fasting Blood Sugar
Resting ECG
Maximum Heart Rate
Exercise Angina
ST Depression (Oldpeak)
ST Slope
🎯 Objective

The goal of this project is to assist in the early prediction of heart disease risk using machine learning techniques and healthcare data.
